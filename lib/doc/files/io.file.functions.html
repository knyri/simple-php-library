file_get_type()

file_get_type(string  $file, string  $default = 'application/octet-stream') : string

Examines the first 256 bytes of a file and attempts to identify it. Returns the common MIME type for the file.

Can currently identify

  • 3gp video/3gpp
  • mov video/quicktime
  • mpeg video/mpeg
  • mpeg2 video/mpeg
  • mp3 audio/mpeg
  • mp4 video/mp4
  • webm video/webm
  • asf container(wmv,wma,asf) video/x-ms-asf
  • mkv container video/x-matroska
  • flv video/x-flv
  • ogg video/ogg
  • avi video/x-msvideo
  • wav audio/x-wav

Parameters

string $file

The file to identify.
string $default

Value returned if the file could not be identified

Returns

string —

Returns the common MIME type for the file.
